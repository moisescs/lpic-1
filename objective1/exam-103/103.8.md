# 103.8 Perform basic file editing operations using vi

## Cursor
* `h` shift for left
* `j` shift for down
* `k` shift for up
* `l` shift for right

## Modos

* `i` entra em modo de inserção.
* `o` cria uma nova linha e entra em modo de inserção.

## Comandos
* `SHIFT + ZZ` = :wq
* `SHIFT + A` insert mode in the end line
* `:e!` reopen
