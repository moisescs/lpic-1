## 102.5 Use RPM and YUM package management

### rpm

```
rpm -i <pkg> (install)
rpm -e <pkg> (remove)
rpm -U <pkg> (update)
rpm -qi <pkg> (info)
rpm -qa (list installed packages)
rpm -ql <pkg> (list files of package)
rpm -qf <pkg> (file is part of what package)
```

### yum

* `/etc/yum.conf`
* `/etc/yum.repos.d`

```
yum install <pkg>
yum remove <pkg>
yum erase <pkg>
yum update <pkg>
yum search <pkg>
yum list <pkg>
```

### rpm2cpio cpio

```
rpm2cpio <pkg> > pkg.cpio
cpio -id < pkg.cpio
```
