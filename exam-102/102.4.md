## 102.4 Use Debian package management

### dpkg dpkg-reconfigure

```
dpkg -i <pkg> (install)
dpkg -r <pkg> (remove)
dpkg -P <pkg> (purge)
dpkg -I <pkg> (info)
dpkg -L <pkg> (list files)
dpkg -l (list installed packages)
dpkg -S <file> (file is part of what package)
dpkg --contents <pkg> (files of the package)

dpkg-reconfigure tz-data
```

### apt

```
apt install <pkg>
apt remove <pkg>
apt purge <pkg>
apt search <pkg>
apt update
apt upgrade
```

### apt-get apt-cache

```
apt-get install <pkg>
apt-get remove <pkg>
apt-get purge <pkg>
apt-cache search <pkg>
apt-get update
apt-get upgrade

apt-cache dump
apt-cache stats
```

### aptitude

```
aptitude install <pkg>
aptitude remove <pkg>
aptitude purge <pkg>
aptitude search <pkg>
aptitude update
aptitude upgrade
```
